
# Python for Data Science Final Assignment

[Link to Github repo](https://github.com/amiengineer/Python-for-Data-Science)

This repository contains the code and final report for a data science project investigating global CO₂ emissions drivers, decarbonization trends, and renewable energy cost forecasts, using datasets from Our World in Data.

## Repository Contents

| File Name | Description |
| :--- | :--- |
| **`01_Biggest_predictor_of_CO2.ipynb`** | An analysis identifying the combination of high economic activity and fossil fuel intensity as the primary predictor of high per-capita CO₂ emissions. |
| **`02_Countries_cutting_CO2_emissions.ipynb`** | A study visualizing which nations have successfully decoupled GDP growth from carbon emissions over the last 15 years using a "Rainbow Chart" methodology. |
| **`03_Best future price for non-fossil fuel energy.ipynb`** | A forecasting model using log-linear regression on historical LCOE data to project future price trends for non-fossil fuel energy sources through 2040. |
| **`Python For Data Science Final Assignment.pdf`** | The comprehensive final report synthesizing the key drivers of emissions, global reduction progress, and future energy pricing trends. |
